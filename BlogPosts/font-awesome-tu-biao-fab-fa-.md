# Font Awesome 图标

最近参考其他人的ejs代码时，发现有些图标使用不是图片。
研究一段时间后发现，是 图标字体库和CSS框架 构建的图标。
大部分图标都能在库里找到，本地可以减少很多不必要本地的图标文件。
